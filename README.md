# Medmart

Medmart is an online store that provides medical and safety standard products.

---

## Project Overview

This landing page is a simple, static website created to showcase a product, service, or brand. 

It includes:

- Navigation bar  
- A responsive hero section  
- Features/Services section  
- Product section   
- FAQ section   
- Contact or footer section  

---

## Technologies Used

- **HTML5** – Page structure  
- **CSS3** – Styling and responsiveness  
- **Bootstrap 5** - Layout, components, and responsiveness
- **Google Fonts** - Typography (3 Google Fonts used)

---

## Setup Instruction

1. Download the ZIP file from the GitHub repository:
[Medmart](https://github.com/binulad/Medmart)
2. Unzip the project folder.
3. Open the `index.html` file in any browser to run the project.
